    Hugo 主题 #

    hugo-academic #

    适合作为个人主页，内容比较丰富，尤其适合作为技术或科研人员的博客。

    hugo-book #

    • 适合记笔记，或者写书。我很喜欢他右侧的页面目录，类似语雀。
    • 其实语雀编辑体验很好，尤其是最近改版后允许用户直接在左侧目录操作新增页面。
    • 不使用语雀，而使用 github pages，最关键也可以说唯一的原因就是对其他平台不放心，内容还是自己保管比较好。
    • 不然万一哪天平台下线，只是一个道歉页面，而作为免费用户，只能自己想办法导出笔记，想想还是算了。

    参考：https://themes.gohugo.io
